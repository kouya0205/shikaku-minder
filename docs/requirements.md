# 資格スケジュール管理サービス「シカクマインダー」要件定義書

## 1. プロダクト概要
「シカクマインダー (Shikaku Minder)」は、資格試験の申込期間・試験日を一元管理し、ユーザーに適切なタイミングでリマインドを行うことで、申込忘れをゼロにするサービス。ミニマルなUIとカレンダー連携を核とし、アフィリエイト送客を軸としたマネタイズを行う。

## 2. ターゲットユーザー
- 国内受験者数トップ30の主要資格（IT、ビジネス、語学など）の取得を目指すユーザー。
- 複数の公式サイトを巡回することにストレスを感じている、効率重視のスマートフォンユーザー。

## 3. 解決する課題 (ペインポイント)
- 資格ごとに異なる、煩雑でレガシーな公式サイトの情報構造。
- 短い申込受付期間の見落としによる、数ヶ月単位の学習期間のロス。
- 複数の資格を並行して検討する際のスケジュール管理コスト。

## 4. コア機能要件 (MVPスコープ)
1. **スマート資格検索**: トップ30資格の最新日程をキーワードやカテゴリから即座に検索。
2. **マイリスト（トラッキング）**: ログイン不要で、気になる資格を「マイリスト」に保存。
3. **シームレス・リマインド**: 
   - Google/Appleカレンダーへのワンタップ登録。
   - ブラウザのWeb Pushを用いた期限通知。
4. **学習リソース送客**: 資格種別に基づいた最適な参考書やオンライン講座（アフィリエイト）の自動提示。

## 5. 非機能要件
- **デザインシステム**: Indigo（信頼・知性）とZinc（モダン・洗練）を基調とした配色。
- **プラットフォーム**: レスポンシブWebアプリケーション。
- **ユーザー認証**: 認証なし（ブラウザの `localStorage` 活用）。

## 6. マネタイズ戦略
- 関連教材のアフィリエイト、学習プラットフォームへの送客、バナー広告。
